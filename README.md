Student Grade Calculator
A simple Python command-line program that collects student scores, calculates their average, assigns letter grades, and gives feedback based on performance.
What It Does
	•	Lets you add one or more students by names
	•	Asks how many subjects each student has, then collects a score for each subject
	•	Validates input so the program doesn’t crash if you type letters instead of numbers
	•	Calculates the student’s average score
	•	Converts each numeric score into a letter grade (A, B, C, D, or F)
	•	Gives feedback based on the student’s worst subject grade (so one weak score still gets flagged, even if the overall average looks good)
	•	Displays a full grade report for each student
	•	Lets you keep adding students until you choose to stop

How to Run
	1.	Make sure Python 3 is installed on your machine.
	2.	Open a terminal in the project folder.
	3.	Run: python3 Student_Grade_Calculator.py and python3 oop_Student_Grade_Calculator.py
    4.	Follow the prompts.
